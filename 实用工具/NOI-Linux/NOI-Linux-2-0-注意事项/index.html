    事实上在前往 NOI 之前，大部分情况下不会遇到只有 NOI LINUX 的电脑。大家普遍使用装载 NOI LINUX 的虚拟机。现在默认使用 VMware Workstation 16 Playdaer。

    虚拟机的使用

    注意 NOI LINUX 的默认密码是 123456。但有些考室会重置密码，等到考试前几分钟才公布。
    这里的文件夹能够使用的就只有 下载、桌面与 csp，csp 文件夹是我们存放比赛数据的地方。如果你的虚拟机中 csp 文件夹被上锁了，在左上角中选择 Player/管理/虚拟机设置/选项/共享文件夹，选择 总是启用，下面的选项是你的共享文件夹在主机中的位置，可以自由设置。

    VSCode 的使用

    然后你就进入了桌面，左下角的九宫格能点进去查看详细应用。事实上在众多应用中能用的没几个。我们选择 VSCode（Visual Studio Code）。点击 “Open Folder”，在文件夹中选择 csp。接下来就是 VSCode 的基本使用了，不会的麻烦去翻一下教程。左侧 CSP 有方的四个 logo 分别代表 创建文件、创建文件夹、刷新、我也不知道干什么用。
    熟悉完基本环境，来看使用。下方小窗口（没有的可以用 Ctrl + ~ 唤起）中，选择 TERMINAL 启用终端（当然也可以使用 Ctrl + Alt + T 打开命令行终端）。
